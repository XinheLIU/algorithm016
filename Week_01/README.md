遇到问题时，首先应该相处几种可能的办法，然后再执行，以下是我总结的破题点

1. 当遇到需要暴力循环的时候，可以考虑快慢指针的方式来节省遍历次数，或者用空间换时间，保存曾经的结果供后面使用。在此应该注意index的标号开始、结束。
2. 计算机最终只能执行循环和判断，所以找出最小重复计算点也是很好的选择
3. 当发现有最近相关性的话可以尝试用栈
